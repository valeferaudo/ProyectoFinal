Preguntas frecuentes

1. ¿Cómo reservar un turno?
Para reservar un turno, es tan sencillo como buscar en canchas y seleccionar la más adecuada a tus preferencias. Luego filtrar la fecha y horario y confirmar el turno. Recuerda que algunos centros deportivos exigen un pago por adelantado.
2. ¿Cómo realizar un pago?
Al momento de confirmar un turno puede realizar el pago total o señar el turno, dependiendo de las especifiaciones del centro deportivo. Una vez realizada la confirmación de un turno, tambien puede realizar el pago, o completar el pago del turno. Los pagos se realizan mediante MercadoPago.
3. ¿Se pueden cancelar turnos reservados?
Los turnos pueden ser cancelados con una anticipación de 12hs. En caso de tener realizado algún pago, deberá contactarse con el centro deportivo correspondiente para obtener el reembolso.
4. ¿Se puede cambiar la contraseña?
La contraseña puede ser cambiada. Para realizar esto, debes:
1- Ingresar a "Editar perfil", ubicado en la esquina superior derecha.
2- Ingresar a "Contraseña".
3- Completar los datos solicitados.
Recuerda que necesitarás indicar la contraseña actual.
5. ¿Qué sucede si no asisto al turno?
Si no asiste al turno, sin previa cancelación o aviso perderá el dinero pagado. Además quedará registrado en el sistema como infractor.
Un usuario marcado como infractor puede llegar a recibir cancelaciones de turnos por parte de los centros deportivos.
